# 即刻用户搜索条件 — 技术型创业者

## 搜索关键词

| 组别 | 关键词 | 目标人群 |
|------|--------|---------|
| 技术作品 | `开源` `GitHub` `hackathon` `黑客松` | 有实际代码产出的人 |
| 创业阶段 | `独立开发` `冷启动` `founder` `PMF` | 正在验证产品的早期 founder |
| 技术方向 | `AI coding` `agent` `LLM` `编译器` | 深度技术方向 |
| 海外背景 | `出海` `硅谷` `Stanford` `MIT` | 有国际视野的技术人 |

## 评分标准（Claude 判断）

| 维度 | 高分信号 |
|------|---------|
| 技术深度 | bio/帖子涉及具体技术栈、开源项目、竞赛成绩 |
| 产品执行力 | 有已发布产品、真实用户数据、变现记录 |
| 成长轨迹 | 帖子记录踩坑/迭代过程，不只是成果展示 |
| 学术背景 | 顶校（国内985/海外T20）或科研/实习经历 |

## 输出表格字段

| 字段 | 说明 |
|------|------|
| 显示名 | screenName |
| 主页链接 | okjike.com/u/username |
| Bio 摘要 | bio 原文（前120字） |
| 推荐理由 | 1-2句，基于 bio 内容判断 |
| 联系方式 | 从 bio 提取微信/Twitter/Email/GitHub，无则留空 |
| 年龄 | 从 bio 提取，无则留空 |
| 触发关键词 | 哪个关键词搜到的 |
